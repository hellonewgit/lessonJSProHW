LeetCode Backend API Documentation

POST /api/users/register

Register a new user

Request Body:

{
    "username": "string",
    "password": "string",
    "role": "string" // optional, defaults to "user"
}

Response:

// Success (201)
{
    "message": "User registered successfully",
    "user": {
        "userid": number,
        "username": "string",
        "role": "string"
    }
}

// Error (400)
{
    "error": "Error message"
}

POST /api/users/login

Login with existing credentials

Request Body:

{
    "username": "string",
    "password": "string"
}

Response:

// Success (200)
{
    "message": "Login successful",
    "user": {
        "userid": number,
        "username": "string",
        "role": "string"
    }
}

// Error (401)
{
    "message": "Authentication failed"
}

GET /users

Get list of all users

Response:

// Success (200)
[
    {
        "userid": number,
        "username": "string",
        "role": "string"
    }
]

GET /task

Get list of all tasks

Response:

// Success (200)
[
    {
        "id_list": number,
        "title": "string",
        "description": "string",
        "difficulty": "string"
    }
]
