Использование API на примере суммы двух чисел.

Эндпоинты API

POST /api/two-sum

Описание: Решает задачу "Two Sum", возвращая индексы двух чисел, которые в сумме дают целевое значение. 

    
                Запрос:
                    Тип: application/json
                    Тело запроса:

                    {
                        "nums": [2, 7, 11, 15],
                        "target": 9
                    }

                Ответ:
                    Успех (200 OK):
                        Тип: application/json
                        Тело ответа:

                        {
                            "indices": [0, 1]
                        }

                    Ошибка (400 Bad Request):
                        Тип: application/json
                        Тело ответа:

                        {
                            "error": "Invalid input. Please provide an array of numbers and a target."
                        }

GET /api/two-sum/example


                Описание: Возвращает пример входных данных для задачи "Two Sum".
                Запрос: Нет
                Ответ:
                    Успех (200 OK):
                        Тип: application/json
                        Тело ответа:

                        {
                            "example": {
                                "nums": [2, 7, 11, 15],
                                "target": 9
                            }
                        }

Примеры использования

Пример 1: Решение задачи "Two Sum"

Запрос:


                curl -X POST http://localhost:3000/api/two-sum \
                -H "Content-Type: application/json" \
                -d '{
                    "nums": [2, 7, 11, 15],
                    "target": 9
                }'

Ответ:


                {
                    "indices": [0, 1]
                }

Пример 2: Ошибка при неверном вводе

Запрос:


                curl -X POST http://localhost:3000/api/two-sum \
                -H "Content-Type: application/json" \
                -d '{
                    "nums": "not an array",
                    "target": 9
                }'

Ответ:


                {
                    "error": "Invalid input. Please provide an array of numbers and a target."
                }

Пример 3: Получение примера входных данных

Запрос:


                curl -X GET http://localhost:3000/api/two-sum/example

Ответ:


                {
                    "example": {
                        "nums": [2, 7, 11, 15],
                        "target": 9
                    }
                }
